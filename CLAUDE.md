# Claude Project Guidance

This project is a small static website for Marjo Seki with Finnish content and a Japanese-inspired visual direction.

## Design direction
- Favor a calm, elegant, and warm Japanese-inspired aesthetic.
- Use soft sakura pinks, muted forest greens, warm cream, and deep burgundy accents.
- Keep typography refined with a serif headline style and a clean sans-serif body style.
- Make the experience feel premium, editorial, and welcoming.

## Implementation notes
- Keep the site lightweight and static.
- Prefer small, polished CSS updates over heavy redesigns.
- Improve button states, section spacing, cards, and readability.
- Preserve the existing structure in HTML and JSON-driven content.
